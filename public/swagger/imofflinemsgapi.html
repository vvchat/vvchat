    Security Definitions

    app_id

    应用ID
    The type of security scheme is apiKey.
    The name of the header to be used is app_id.

    Authorization

    用户token
    The type of security scheme is apiKey.
    The name of the header to be used is Authorization.

    sign

    签名
    The type of security scheme is apiKey.
    The name of the header to be used is sign.

    noncestr

    随机字符串
    The type of security scheme is apiKey.
    The name of the header to be used is noncestr.

    timestamp

    时间戳
    The type of security scheme is apiKey.
    The name of the header to be used is timestamp.

    获取离线消息

    获取用户的离线消息

    /users/{open_id}/offlinemsgs
